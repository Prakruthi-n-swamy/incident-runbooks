# incident-runbooks
Production incident runbooks, RCA templates, troubleshooting guides and operational playbooks for common DevOps scenarios.


## Included Runbooks

- SQS Queue Backlog
- API Latency Spike
- Linux Disk Full
- Deployment Rollback
- Vulnerability Remediation

## Documentation Templates

- Root Cause Analysis
- Incident Reports
- Postmortems

## Purpose

Standardize operational response procedures and improve incident resolution consistency.

## Incident Lifecycle

```text
Alert
  ↓
Investigation
  ↓
Mitigation
  ↓
Resolution
  ↓
RCA
  ↓
Prevention
```
