# API Latency Spike Runbook

## Symptoms

- Increased response times
- SLA degradation
- AppDynamics alerts

## Investigation

1. Check AppDynamics transactions
2. Review server resource utilization
3. Analyze application logs
4. Validate dependent services

## Resolution

- Restart impacted service
- Resolve dependency bottlenecks
- Scale infrastructure if required

## Prevention

- Performance testing
- Capacity planning
- Early warning alerts
